# Meals That Connect

A web-based recipe and menu management platform built for Meals That Connect, a nonprofit that serves over 1,000 meals a day to seniors across San Luis Obispo County.

## Overview

This project was developed in collaboration with **Meals That Connect** through Cal Poly’s Hack4Impact program. It replaces a spreadsheet- and paper-based system with a centralized web platform that enables:

- Recipe management
- Menu planning
- Role-based access for nonprofit staff, volunteers, and public users

The platform is designed to integrate directly with the organization’s Google Drive, preserving their current data structure while modernizing their workflows.

### Purpose

**Meals That Connect** is a nonprofit organization dedicated to enhancing the health, dignity, and independence of adults aged 60 and over in San Luis Obispo County by providing free, hot, noon-time meals through home delivery and community dining sites.

The purpose of this project is to digitize and streamline the nonprofit’s existing processes for managing recipes and planning menus. The current system relies heavily on Google Sheets and manual, repetitive work, which can lead to errors and inefficiencies.

By building this platform, we aim to:

- Save time by eliminating redundant data entry
- Improve data accuracy across recipes and nutritional information
- Support staff and volunteers with centralized, accessible tools
- Enable seniors to easily view current and upcoming menus

This system will directly help Meals That Connect scale its services and better serve its senior community.
